# ProjectSensiY

Criado em 2020, o software tem como objetivo melhorar a sensibilidade de seus usuários, muito utilizado para jogadores
do game FREE FIRE, porém 100% funcional para outros jogos, compatível com
todos os emuladores existentes, para quem tem um mouse genérico sem dpi,
o PSY pode estar lhe ajudando ao aumento de sua sensibilidade do eixo Y,
várias outras versões foram publicadas, porém, esta versão possui uma pacote de configuração
 e criptografia  totalmente diferente das outras versões que eram BETA, tendo a 
sensibilidade 100% otimizada para todos os emuladores.
Claro que você precisa configurar juntamente a sua jogabilidade
 do  game que estará utilizando, não adianta você ativar a sensi do eixo y e
não configurar o game em que vai utilizar.
